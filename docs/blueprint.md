# **App Name**: SETU Connect

## Core Features:

- User Authentication: Secure login and authentication system for patients, first responders, hospital staff, and administrators.
- Patient Interface: Dedicated interface for patients to request assistance and view their status.
- First Responder Interface: Interface for first responders to receive alerts, view patient information, and update their status.
- Hospital Staff Interface: Interface for hospital staff to manage incoming alerts, patient information, and assign resources.
- Admin Dashboard: Administrative dashboard for managing users, permissions, and system configurations.
- Emergency Alert System: Real-time alert system for critical events, providing location and patient data.
- Fake/Dummy Database Integration: Integration with a mock database to simulate data flow and functionality for demonstration and testing purposes.

## Style Guidelines:

- Primary color: Calm blue (#4A8FE7) to evoke trust and stability.
- Background color: Light gray (#F0F4F8) for a clean and modern look.
- Accent color: Vibrant orange (#FF7A00) for emergency alerts and calls to action.
- Body and headline font: 'Inter' (sans-serif) for clear readability and a modern feel.
- Use clear and intuitive icons to represent different functionalities and alerts.
- Prioritize a clean, responsive layout to ensure ease of use on various devices.
- Subtle animations to guide user interactions and provide feedback during critical processes.
